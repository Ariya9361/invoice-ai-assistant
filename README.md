# Invoice AI Assistant

## Project Info

**Live URL:** https://invoice-ai-assistant-six.vercel.app/

Invoice AI Assistant is a web application that automates invoice data extraction and validation using AI concepts. Users can upload invoices, extract structured information, and streamline invoice processing workflows.

---

## How to Edit This Code

You can edit the project using your preferred IDE.

### Local Development Setup

Make sure Node.js and npm are installed.

Follow these steps:

```sh
# Clone repository
git clone <YOUR_GIT_URL>

# Navigate into project
cd invoice-ai-assistant

# Install dependencies
npm install

# Start development server
npm run dev
